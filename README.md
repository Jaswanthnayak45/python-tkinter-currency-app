#  Real-Time Currency Converter (Desktop GUI)

A professional, high-performance desktop application built with **Python** and **Tkinter** that provides real-time currency conversion rates for **160+ global currencies**. This project demonstrates the integration of RESTful APIs with a modern Graphical User Interface (GUI).

---

## Key Features

* **Real-Time Rates:** Fetches live exchange data using the **ExchangeRate-API**.
* **160+ Currencies:** Supports all major global currencies (USD, INR, EUR, GBP, etc.).
* **Instant Swap:** One-click "Swap" button to invert the *From* and *To* currencies.
* **Robust Error Handling:** Managed exceptions for network connectivity and invalid currency codes.
* **Modern UI:** Clean, professional dark-themed interface designed with Tkinter.
* **Zero-Setup API:** Uses a public endpoint (v4) to ensure immediate functionality without complex API key configurations.

---

## Tech Stack

* **Language:** Python 3.x
* **Library:** Tkinter (Standard GUI Library)
* **Networking:** `requests` library for HTTP API calls.
* **API:** [ExchangeRate-API](https://www.exchangerate-api.com/)
